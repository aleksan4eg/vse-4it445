# VSE Backend

## Run On Server

Use SSH to login to server and run this:

```
cd code/cviceni/backend
yarn install
yarn build:watch
```

Now you can edit code in Atom. To stop editing press <key>Ctrl</key>+<key>C</key> and log off SSH.

### Run Locally

Open terminal an run:

```
cd backend
yarn install
yarn dev
```
